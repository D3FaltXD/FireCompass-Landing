## 📖 Overview

This repository contains a collection of Cross-Site Scripting (XSS) Proof of Concepts (POCs) for educational and security testing purposes. XSS is a type of security vulnerability that allows attackers to inject malicious scripts into web pages viewed by other users.

⚠️ **DISCLAIMER: These POCs are for educational purposes only. Only use them on systems you own or have explicit permission to test.**

## 🔍 What's Inside

- Various HTML files demonstrating different XSS techniques
- Login form vulnerabilities
- DOM-based XSS examples
- Reflected XSS demonstrations
- Stored XSS scenarios
- Bypass techniques for common XSS filters
